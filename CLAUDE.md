# European Cultural Mosaic

A web project exploring the architecture, cuisine, art, and lifestyle of 6 European countries: France, Italy, Spain, Germany, United Kingdom, and Greece.

## Tech Stack
- Pure HTML and CSS only. No JavaScript, no frameworks, no external APIs.
- Google Fonts are allowed via `<link>` tags.

## Structure
- `index.html` — Gallery page showing all 25 versions
- `styles/gallery.css` — Gallery page styles
- `versions/v1.html` through `versions/v25.html` — Self-contained landing pages with inline `<style>` blocks

## Style Preferences
- Clean, well-structured HTML5
- CSS custom properties for color systems
- Responsive design (mobile-first when possible)
- Each version file should be fully self-contained
